# Crescere Curriculum Documentation

## Purpose

This folder contains internal curriculum planning and teaching materials for Crescere language courses. These documents are for instructor use, program planning, and future student-facing materials. They are **not** imported into the public website and should not be copied directly to course pages without a separate review and editing pass.

## Program Structure

| Parameter | Value |
|---|---|
| Program length | 12 weeks |
| Sessions per week | 2 |
| Session length | 1 hour |
| Total live sessions | 24 |
| Homework per session | ≤30 minutes |
| Cohort size | 5–12 students |
| Platform | Google Meet |
| Languages | French, Spanish |
| Levels | A1, A2, B1 |

## CEFR Level Alignment

Courses map directly to the Common European Framework of Reference for Languages (CEFR):

- **A1 (Beginner):** Absolute beginners building foundational grammar and vocabulary
- **A2 (Elementary):** Students who can manage present tense and basic exchanges; cannot yet discuss the past reliably
- **B1 (Intermediate):** Students who can handle past, present, and future; cannot yet use subjunctive, conditional, or complex structures confidently

## How to Use These Files

- **Before teaching a course:** Read the course file end-to-end. Flag any session that looks too ambitious for 60 minutes and adjust.
- **When planning sessions:** Use the session-by-session breakdown as a starting point, not a script. Adjust pacing based on cohort needs.
- **When updating the website:** Do not copy curriculum text directly to public pages. Website copy should be marketing-appropriate summaries, not teaching plans.
- **When designing homework:** Use the approved ≤30-minute guideline. If a homework assignment risks running long, provide a shorter alternative or split it.
- **When evaluating student readiness:** Use the "Readiness for next level" criteria in each course file, not informal impression.

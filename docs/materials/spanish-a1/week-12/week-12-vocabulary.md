| Spanish | English | Verb |
|---|---|---|
| *Soy de [city/country].* | I'm from [place]. | ser — origin |
| *Soy [nationality/profession].* | I'm [nationality/profession]. | ser — identity |
| *Soy alto/a, moreno/a...* | I'm tall, dark-haired... | ser — permanent description |
| *Estoy cansado/a / contento/a.* | I'm tired / happy. | estar — state |
| *Estoy en casa / en el trabajo.* | I'm at home / at work. | estar — location |
| *La farmacia está al lado del banco.* | The pharmacy is next to the bank. | estar — location of known thing |
| *Hay una farmacia en esta calle.* | There is a pharmacy in this street. | hay — existence |
| *Llevo [time] estudiando español.* | I've been studying Spanish for [time]. | llevar — duration |

## Quick Grammar Reference for the Final Performance

### Ser vs. Estar (Week 2)
- **Ser:** identity, origin, profession, permanent description — *Soy inglés, soy médico, soy alto*
- **Estar:** states, emotions, location — *Estoy cansado, estoy en casa, el banco está enfrente*

### Ir a + Infinitivo (Week 9)
*voy / vas / va / vamos / vais / van* + a + infinitivo
- *Voy a estudiar esta tarde.*

### Key Obligation Structures (Week 9)
- *Tener que + inf* (personal): *Tengo que trabajar*
- *Hay que + inf* (general): *Hay que estudiar*
- *Poder + inf* (ability): *Puedo ayudarte*

### Gustar / Doler (Weeks 6, 11)
- *me gusta / me gustan* — I like...
- *me duele / me duelen* — my... hurts/hurt (body part = subject; definite article, not possessive)

### Llevar + Time (Week 11)
- *Llevo [time] + gerund*: *Llevo una semana estudiando*
- *Llevo [time] + con + noun*: *Llevo tres días con fiebre*

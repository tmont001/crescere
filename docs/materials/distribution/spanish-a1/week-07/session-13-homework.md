## Part 1 — Demonstratives: Este or Ese?

Supply the correct demonstrative adjective. Use *este/esta/estos/estas* for things near you and *ese/esa/esos/esas* for things farther away. The context tells you which to use.

Write the correct *este* form for each noun:

1. _____ pantalón (m.)
2. _____ camisa (f.)
3. _____ zapatos (m. pl.)
4. _____ botas (f. pl.)
5. _____ vestido (m.)
6. _____ chaqueta (f.)

Now write the correct *ese* form for the same nouns:

7. _____ pantalón
8. _____ camisa
9. _____ zapatos
10. _____ botas

**Answer key:**
1. este  2. esta  3. estos  4. estas  5. este  6. esta
7. ese  8. esa  9. esos  10. esas

## Part 2 — Color Agreement (5 sentences)

Write each phrase with the color correctly agreeing with the clothing noun. Check your Week 7 Vocabulary Reference for genders if needed.

1. una camisa + blanco
2. un pantalón + negro
3. unos zapatos + marrón
4. una chaqueta + verde
5. unas zapatillas + naranja *(invariable!)*

**Answer key:**
1. una camisa **blanca**
2. un pantalón **negro**
3. unos zapatos **marrones** *(marrón → marrones in plural)*
4. una chaqueta **verde** *(verde: same for m./f.; verde → verdes in plural — here it's singular: verde)*
5. unas zapatillas **naranja** *(invariable — never changes)*

## Part 3 — Shopping Dialogue (6 lines minimum)

Write a short dialogue between a customer (*C*) and a shop assistant (*D* for dependiente/a). You are the customer.

**Required elements:**
- [ ] Customer states what they're looking for: *Busco...*
- [ ] Customer uses a demonstrative: *este/esta/ese/esa + [item]*
- [ ] Customer asks about a color or size: *¿Tiene... en [color], talla [M/L]?*
- [ ] Customer asks to try it on: *¿Me la/lo puedo probar?*
- [ ] Customer asks the price: *¿Cuánto cuesta?*
- [ ] Customer buys or declines: *Me la/lo llevo* or *No, gracias*

---

## Part 4 — Speaking Practice

Read your dialogue from Part 3 out loud. Say both parts. Third try: no notes on the core shopping phrases (*Busco..., ¿Me la puedo probar?, ¿Cuánto cuesta?*).

Then describe what you are wearing right now in Spanish. Use at least 3 items with colors: *Llevo unos vaqueros azules y una camiseta blanca...*

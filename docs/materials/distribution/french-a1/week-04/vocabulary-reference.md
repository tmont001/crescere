## Reflexive Pronouns

Reflexive verbs use an extra pronoun between the subject and the verb. The pronoun refers back to the subject — the action is done *to* or *for* oneself.

| Subject | Reflexive pronoun | Before vowel |
|---|---|---|
| je | me | m' |
| tu | te | t' |
| il / elle | se | s' |
| nous | nous | nous |
| vous | vous | vous |
| ils / elles | se | s' |

**Position:** The reflexive pronoun goes directly before the conjugated verb.
- *Je me lève.* — I get up.
- *Tu te lèves.* — You get up.
- *Il se lève.* — He gets up.

**Before a vowel or silent h:** *me → m', te → t', se → s'*
- *Je m'appelle Marie.* (not: *Je me appelle*)
- *Elle s'habille rapidement.* (not: *Elle se habille*)

## Core Reflexive Verbs — Daily Routine

| Infinitive | Meaning | Je form | Example sentence |
|---|---|---|---|
| s'appeler | to be called | je m'appelle | *Je m'appelle Sophie. Comment vous appelez-vous ?* |
| se réveiller | to wake up | je me réveille | *Je me réveille à six heures et demie.* |
| se lever | to get up | je me lève | *Je me lève vers sept heures.* |
| se doucher | to shower | je me douche | *Je me douche le matin.* |
| se laver (les mains / les dents) | to wash (hands / teeth) | je me lave | *Je me lave les dents après le petit-déjeuner.* |
| s'habiller | to get dressed | je m'habille | *Je m'habille rapidement.* |
| se préparer | to get ready | je me prépare | *Je me prépare en vingt minutes.* |
| se coucher | to go to bed | je me couche | *Je me couche vers onze heures.* |

**Note on se lever:** The *-e-* in the stem becomes *-è-* in all forms except *nous* and *vous* — *je me lève, tu te lèves, il se lève,* but *nous nous levons, vous vous levez.* This is an accent change, not a meaning change. Use it as a phrase for now — you do not need to memorize this pattern yet.

## Time Expressions — Telling Time (Introduction)

| French | English |
|---|---|
| Il est une heure. | It is one o'clock. |
| Il est deux heures. | It is two o'clock. |
| Il est midi. | It is noon. |
| Il est minuit. | It is midnight. |
| Il est deux heures et demie. | It is half past two. |
| Il est trois heures et quart. | It is quarter past three. |
| Il est quatre heures moins le quart. | It is quarter to four. |
| À quelle heure...? | At what time...? |
| vers... | around... (time) |
| à + heure | at + time |

**Using time with routine verbs:**
- *Je me réveille à six heures et demie.*
- *Je me couche vers onze heures.*

## -ir and -re Verbs — Light Preview (Extension)

These are not Week 4 core content. For reference and motivated learners:

**Regular -ir verbs (finir pattern):**

| Subject | Finir (to finish) |
|---|---|
| je | finis |
| tu | finis |
| il / elle | finit |
| nous | finissons |
| vous | finissez |
| ils / elles | finissent |

Other common -ir verbs: *choisir* (to choose), *réussir* (to succeed), *réfléchir* (to think/reflect)

**Regular -re verbs (répondre pattern):**

| Subject | Répondre (to answer) |
|---|---|
| je | réponds |
| tu | réponds |
| il / elle | répond |
| nous | répondons |
| vous | répondez |
| ils / elles | répondent |

Other common -re verbs: *vendre* (to sell), *entendre* (to hear), *attendre* (to wait)

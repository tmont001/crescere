## Part 1 — Numbers 31–100: The Patterns (10 min)

You learned 1–30 in class and Session 1 homework. Here is the rest of the system.

### Regular tens: 30–69

Simple pattern: tens + *et* (for ×1) or tens + unit.

| | | |
|---|---|---|
| 30 — trente | 40 — quarante | 50 — cinquante |
| 60 — soixante | | |

**Examples:** 35 = *trente-cinq* · 42 = *quarante-deux* · 61 = *soixante et un* · 68 = *soixante-huit*

### 70–79 — Counted as "sixty-and-ten"

French treats 70 as 60+10. There is no word for seventy.

| | |
|---|---|
| 70 = soixante-dix (60+10) | 75 = soixante-quinze (60+15) |
| 71 = soixante et onze (60+11) | 76 = soixante-seize (60+16) |
| 72 = soixante-douze (60+12) | 77 = soixante-dix-sept (60+17) |
| 73 = soixante-treize (60+13) | 78 = soixante-dix-huit (60+18) |
| 74 = soixante-quatorze (60+14) | 79 = soixante-dix-neuf (60+19) |

### 80–89 — "Four twenties"

80 = *quatre-vingts* (4×20). After 80: drop the -s and add units.

| | |
|---|---|
| 80 = quatre-vingts | 85 = quatre-vingt-cinq |
| 81 = quatre-vingt-un | 86 = quatre-vingt-six |
| 82 = quatre-vingt-deux | 87 = quatre-vingt-sept |
| 83 = quatre-vingt-trois | 88 = quatre-vingt-huit |
| 84 = quatre-vingt-quatre | 89 = quatre-vingt-neuf |

### 90–99 — "Four twenties and ten"

90 = *quatre-vingt-dix* (4×20+10). Same pattern continues.

| | |
|---|---|
| 90 = quatre-vingt-dix | 95 = quatre-vingt-quinze |
| 91 = quatre-vingt-onze | 96 = quatre-vingt-seize |
| 92 = quatre-vingt-douze | 97 = quatre-vingt-dix-sept |
| 93 = quatre-vingt-treize | 98 = quatre-vingt-dix-huit |
| 94 = quatre-vingt-quatorze | 99 = quatre-vingt-dix-neuf |

### 100

100 = *cent*

## Part 2 — Written Introduction (8 min)

Write five sentences in French introducing yourself. Use the phrases from Sessions 1 and 2.

Your sentences must include:
1. Your name (*Je m'appelle...*)
2. Where you are from (*Je suis de...*)
3. Where you live now (*J'habite à...*)
4. Your profession (*Je suis...*)
5. Your nationality (*Je suis...*)

**Example:**
> *Je m'appelle Claire. Je suis de Montréal. J'habite à New York. Je suis architecte. Je suis canadienne.*

Write yours in your notebook or document.

---

## Part 3 — Speaking Practice (5 min)

Say your written introduction out loud three times. On the third try, close your notes and say it from memory. It should take under 30 seconds.

Then answer these questions out loud, as if someone is asking you:

- *Tu t'appelles comment ?* → *Je m'appelle...*
- *Tu es d'où ?* → *Je suis de...*
- *Tu habites où ?* → *J'habite à...*
- *Tu fais quoi dans la vie ?* → *Je suis...*

## Optional Extension

If you finish early:

**Write about someone else.** Use *il/elle est* to describe a real or invented person — the form you saw in Session 2. For each person, write:
- *Il/Elle s'appelle...*
- *Il/Elle est de...*
- *Il/Elle habite à...*
- *Il/Elle est [profession]. Il/Elle est [nationality].*

**Listen:** Find a short clip of someone introducing themselves in French. Can you catch their name, where they're from, and what they do? You will not understand everything — that is normal. Just notice what you can pick out.

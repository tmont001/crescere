# French A1 — Week 2 Instructor Notes

**Sessions:** 3 and 4
**Theme:** Avoir, Physical Description, and Personality
**Week objective:** Students can describe themselves and one other person — appearance, age, and 3–5 personality traits — with correct adjective agreement on regular patterns.

---

## What Must Be Protected in Live Class Time

- **Avoir — spoken production.** Students need to say all six forms and use *j'ai X ans* before they leave Session 3. Especially *nous avons / vous avez / ils ont* — these are the forms that disappear if not drilled live.
- **The J'ai vs. Je suis contrast.** *J'ai les yeux bleus* (avoir for physical features) vs. *Je suis grand* (être for adjectives used directly). Students will mix these up constantly in Week 2. Model the correct choice explicitly each time.
- **Adjective agreement for gender** — the feminine -e pattern must be heard and practiced live. Seeing it on paper is not enough. Students need to produce *Il est grand / Elle est grande* out loud with your correction.
- **The full description round in Session 4.** Every student must describe a real person out loud at the end of Session 4. This is the week's speaking milestone — protect it the same way you protected the introduction round in Week 1.

## Pacing Guidance

### Session 3 (60 min)

| Block | Content | Time |
|---|---|---|
| Warm-up | Review Week 1 — trigger a quick full introduction round | 5 min |

## Common Student Mistakes — Week 2

| Mistake | Notes |
|---|---|
| *Je suis les yeux bleus* (using être for features) | Contrast clearly: *J'ai les yeux bleus* (feature you have) vs. *Je suis grand* (adjective describing you). The avoir/être split trips students all week. |
| *J'ai vingt-cinq* without *ans* | Age requires *ans* — *J'ai vingt-cinq ans.* Remind each time a student drops it. |
| Forgetting to feminize adjectives | *Elle est grande*, not *Elle est grand.* Model both versions in the same sentence where possible: "If we're talking about Thomas: il est grand. If we're talking about Marie: elle est grande." |
| *Les yeux marrons* (adding -s to invariable) | Demonstrate once that marron and orange never change: *les yeux marron, une robe orange.* Do not over-explain why. |
| *Ils sont grand* (plural agreement missing -s) | Note: *grands* — though plural agreement in speech is usually inaudible, train students to write it correctly. |
| *Sérieuse* written as *sérieuxe* | The -eux → -euse pattern changes the entire suffix, not just adding -e. Show the pattern explicitly. |

## Suggested Instructor Language

- **For avoir:** *"We have être from last week — 'to be.' Now we're adding avoir — 'to have.' In French, age is something you have, not something you are. So instead of 'I am thirty,' you say 'J'ai trente ans' — literally, 'I have thirty years.'"*
- **For J'ai vs. Je suis:** *"Here's the rule of thumb for this week: if you're talking about something you literally possess — eyes, hair — use avoir. If you're describing what you're like — tall, funny — use être."*
- **For adjective agreement:** *"I know this feels like extra work. Here's why it matters: in French, an adjective is supposed to match the noun it describes — like a photograph fitting a frame. When the frame is feminine, the photo changes. Il est grand, elle est grande. You'll hear me correct this every time until it becomes automatic."*
- **For the description round:** *"You now know enough to describe someone in French. I want each of you to describe a real person — a classmate, a family member, a famous person — in four sentences. I'll go first."*

## What to Skip If Time Is Tight

- Hair texture (*lisse, frisé, ondulé*) — mention it exists; it is in the vocabulary reference; do not drill it live
- Full color adjective agreement rules — demonstrate the invariable cases (*marron, orange*) once; teach the full agreement pattern in Session 4 when adjective agreement is the main topic
- BAGS adjective placement — skip entirely in Week 2 unless a student specifically asks; it is not needed to describe people accurately at A1
- Additional personality adjectives beyond the 8 core — the reference list is in the vocabulary sheet; students can explore independently
- *Il/Elle a l'air...* (he/she seems/looks) — Reference phrase; not needed for the Week 2 speaking task

## Week 2 Priority Reminder

The most important thing this week is that students begin **agreeing adjectives** — hearing and producing the masculine/feminine distinction. Accuracy on this pattern will pay dividends for the rest of A1. Correct it consistently and positively every time. Students who skip the habit in Week 2 will produce agreement errors through Week 12.

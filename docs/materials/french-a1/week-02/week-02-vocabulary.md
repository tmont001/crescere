**Key use this week:** *avoir* for age — *J'ai trente ans.* (I am thirty years old.) Note: French uses "have years," not "be years."

## Colors as Adjectives (Core)

Most colors agree in gender and number. Exceptions noted.

| French | English | Feminine | Plural (m) | Invariable? |

| noir | black | noire | noirs | No |
| gris | grey | grise | gris | No |
| bleu | blue | bleue | bleus | No |
| vert | green | verte | verts | No |

| jaune | yellow | jaune | jaunes | No |
| rose | pink | rose | roses | No |
| violet | purple | violette | violets | No |

## Adjective Agreement Summary (Core)

| | Masculine | Feminine | Note |
|---|---|---|---|
| Regular | grand | grande | add -e |
| Already ends in -e | drôle | drôle | no change |
| -eux → -euse | nerveux | nerveuse | pattern change |
| -if → -ive | sportif | sportive | pattern change |
| -el → -elle | naturel | naturelle | double l |
| -il → -ille | gentil | gentille | double l |
| Invariable | sympa, marron, orange | — | never changes |

**Plural:** Add -s to masculine or feminine form. If the word already ends in -s or -x, no change. *Il est nerveux → Ils sont nerveux.*

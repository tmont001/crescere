# French A1 — Week 12 Phrase Bank
## Final Performance Support Language

**Purpose:** This is not a new vocabulary list. It is a curated collection of the most useful phrases from across French A1, organized by the scenarios you will use in the final performance. Use this as a reference during Session 23 practice and Session 24 preparation.

## Quick Grammar Reference for the Final Performance

### Être (Week 1)
*je suis, tu es, il/elle est, nous sommes, vous êtes, ils/elles sont*

### Avoir (Week 2)
*j'ai, tu as, il/elle a, nous avons, vous avez, ils/elles ont*

### Aller + infinitif (Week 9)
*je vais, tu vas, il/elle va, nous allons, vous allez, ils/elles vont* + infinitif

### Modals (Weeks 6, 9)
- *devoir:* dois / dois / doit / devons / devez / doivent
- *pouvoir:* peux / peux / peut / pouvons / pouvez / peuvent
- *vouloir:* veux / veux / veut / voulons / voulez / veulent

### Partitives (Week 6)
*du, de la, de l', des* → after negation: *de*

### Article contractions
- *à + le = au; à + les = aux*
- *de + le = du; de + les = des*

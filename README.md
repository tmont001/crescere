# Crescere — Online Tutoring Platform

A marketing and enrollment site for an online tutoring business, covering the full visitor journey from discovery to enrollment.

## Features

- Marketing pages: Home, About, FAQ, Pricing, Contact
- Course catalog with individual course detail pages
- Placement assessment flow
- Enrollment flow
- Student dashboard (in progress)

## Tech Stack

- **Framework:** React + Vite + TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Animation:** Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/

  components/
  layouts/

  hooks/
  lib/

```
